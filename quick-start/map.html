地图和地形 #

地图大小 (Width / Height) #

注意

根据编写者的习惯，这里的 width 为地图的竖向格子数，height 为地图的横向格子数（（

取值：0∼1。

计算公式：

valw=⌊kings×2.7+10×width⌋

其中 kings 为玩家数量。 height 计算公式类似。

地形 (Terrain) #

一共有 King、Plain、Mountain、City、 Swamp 五类。

其中 Mountain、City、 Swamp 生成参数可调。

Mountain 计算公式：

valm=⌊14×size×mountainmountain+city⌋

City 计算公式：

valc=⌊16×size×citymountain+city⌋

特别地，当 mountain+city=0 时，valm=valc=0

Swamp 计算公式：

vals=⌊13(size−valm−valc)×swamp⌋

其中 size=valw×valh
